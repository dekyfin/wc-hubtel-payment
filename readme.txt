== Description ==
Hubtel Woocommerce Payment Gateway is a great plugin that seamlessly allow you to accept payment from your woocommerce store. Hubtel Payments supports the following payment methods; Mobile Money and Credit/Debit Cards. Your customers will be able to pay for products/services using Credit/Debit Card or Mobile Money.
To signup for a merchant account click [here](https://unity.hubtel.com/account/signup)
**Note**: This plugin is meant to be used by merchants in Ghana.


== Installation ==
1. Login to your WordPress Admin area
2. Go to "Plugins > Add New" from the left hand menu
3. In the search box type "Hubtel Woocommerce Payment Plugin"
4. From the search result you will see "Hubtel Woocommerce Payment Plugin" click on "Install Now" to install the plugin
5. A popup window will ask you to confirm your wish to install the Plugin.
6. Click "Proceed" to continue the installation. The resulting installation screen will list the installation as successful or note any problems during the install.
7. Click "Activate Plugin" to activate it.
8. Open the settings page for WooCommerce and click the "Payment Gateways," tab.
9. Click on the sub tab for "Hubtel Payment Gateway".
10. Configure your "Hubtel Payment settings. See below for details.


== Configure the plugin ==
To configure the plugin, go to **WooCommerce > Settings**Â from the left hand menu, then click "Payment Gateways" from the top tab. You should see **Hubtel Payment Gateway** as an option at the top of the screen. Click on it to configure the payment gateway.
1. Enable/Disable - check the box to enable or disable Hubtel Payment Gateway.
2. Title - allows you to determine what your customers will see this payment option as on the checkout page.
3. Description - controls the message that appears under the payment fields on the checkout page. Here you can list the types of cards you accept.
4. Client Id  - enter your hubtel client id
5. Secret  - enter your hubtel secret
6. Click on **Save Changes** for the changes you made to be effected.
